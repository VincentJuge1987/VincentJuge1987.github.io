Exercice 1 : Hello World

  1. On utilise Ctrl + Maj + K avec Firefox, et Ctrl + Maj + J avec Chrome.
    Tant qu'on n'affiche pas la console, on voit juste le texte "Premier pas en JavaScript".
    Une fois qu'on affiche la console, on voit "hello" écrit dans la console.
  2. Je commente le code console.log("hello"); et je le remplace par 
         for(let i = 0; i <= 2022; i++) {
       console.log(i);
     }
  3. Au lieu du code précédent, j'écris 
         for(let i = 0; i <= 2022; i++) {
       if (i % 3 != 0) {
         console.log(i);
       }
     }
    afin de ne pas afficher les multiples de 3.
  4. Je commente mon code précédent, par exemple à l'aide de /* … */, puis j'écris 
         const showUntil = function(bound) {
       for (let i = 0; i < bound; i++) {
         console.log(i);
       }
     }
     showUntil(3);
     showUntil(12);
    ce qui me permet d'afficher les nombres de 0 à 3 puis de 0 à 12.

Exercice 2 : Damier

  1. "use strict"; permet d'utiliser la syntaxe stricte du JavaScript, plutôt que d'écrire n'importe quoi.
  2. L'appel à document.write() permet d'écrire directement dans le document.
  3. J'écris 
         const drawCheckerLine = function() {
       document.write('<div class="line">');
       for (let i = 0; i < 5; i++) {
         drawCheckerCell();
       }
       document.write('</div>');
     };
    dans la partie dévolue au code Javascript, et 
         div.line {
       padding: 0px;
     }
    dans la partie dévolue au style CSS.
  4. J'écris 
         const drawCheckerBoard = function() {
       document.write('<div class="board">');
       for (let i = 0; i < 5; i++) {
         drawCheckerLine();
       }
       document.write('</div>');
     };
    dans la partie dévolue au code Javascript, et je remplace le style de div.line par 
         div.line, div.board {
       clear: left;
       padding: 0px;
     }
    dans la partie dévolue au style CSS.
  5. J'ajoute maintenant un argument size à mes fonctions drawCheckerLine et drawCheckerBoard, de sorte que mon code est désormais 
         const drawCheckerLine = function(size) {
       document.write('<div class="line">');
       for (let i = 0; i < size; i++) {
         drawCheckerCell();
       }
       document.write('</div>');
     };
     
     const drawCheckerBoard = function(size) {
       document.write('<div class="board">');
       for (let i = 0; i < size; i++) {
         drawCheckerLine(size);
       }
       document.write('</div>');
     };
    et j'en profite pour décommenter le second paragraphe et ajouter 
         p, br {
       clear: left;
     }
    dans la partie dévolue au style CSS.
  6. J'ajoute 
         .black {
       background-color: black;
     }
     .white {
       background-color: white;
     }
    dans la partie dévolue au style CSS. Puis j'ajoute un argument color à ma fonction drawCheckerCell, de sorte que mon code est désormais 
         const drawCheckerCell = function(color) {
       document.write(`<div class="cell ${color}"></div>`);
     };
     
     const drawCheckerLine = function(i, size) {
       document.write('<div class="line">');
       for (let j = 0; j < size; j++) {
         drawCheckerCell((i + j) % 2 == 0 ? 'black' : 'white');
       }
       document.write('</div>');
     };
     
     const drawCheckerBoard = function(size) {
       document.write('<div class="board">');
       for (let i = 0; i < size; i++) {
         drawCheckerLine(i, size);
       }
       document.write('</div>');
     };
  7. J'ajoute 
         .red {
       background-color: red;
     }
    dans la partie dévolue au style CSS. Puis j'ajoute un argument red à mes fonction drawCheckerBoard et drawCheckerLine, de sorte que mon code est désormais 
         const drawCheckerLine = function(i, size, red) {
       document.write('<div class="line">');
       for (let j = 0; j < size; j++) {
         drawCheckerCell((red && i == j) ? 'red' : (i + j) % 2 == 0 ? 'black' : 'white');
       }
       document.write('</div>');
     };
     
     const drawCheckerBoard = function(size, red) {
       document.write('<div class="board">');
       for (let i = 0; i < size; i++) {
         drawCheckerLine(i, size, red);
       }
       document.write('</div>');
     };
  8. Je constate effectivement que la diagonale du premier carré est rouge, et que la deuxième ne l'est pas.

Exercice 3 : On fera en fonction…

  1. Ma nouvelle fonction drawCheckerLine s'écrit 
         const drawCheckerLine = function(size, fun) {
       document.write('<div class="line">');
       for (let j = 0; j < size; j++) {
         drawCheckerCell(fun(j));
       }
       document.write('</div>');
     };
  2. Ma nouvelle fonction drawCheckerBoard s'écrit 
         const drawCheckerBoard = function(size, fun) {
       document.write('<div class="board">');
       for (let i = 0; i < size; i++) {
         drawCheckerLine(size, j => fun(i,j));
       }
       document.write('</div>');
     };
  3. Ma fonction diagonalColor s'écrit 
         const diagonalColor = function(hasRed) {
       return (i,j) => (hasRed && i == j) ? 'red' : ((i + j) % 2 == 0) ? 'black' : 'white';
     };
