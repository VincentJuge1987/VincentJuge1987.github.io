Premiers pas en Javascript
Comme pour les TPs précédents, on vous demande de créer un répertoire tp5 qui contiendra les exercices de cette feuille de TP et de faire un compte rendu de TP en HTML.

Exercice 1 - Hello World

Cet exercice montre comment utiliser les outils de developpement présents sur les navigateurs, comme la console.

  1. Recopiez la page Web exo1.html dans votre répertoire tp5.
    Que se passe-t-il lorsque l'on visualise la page ?
    Comment fait-on pour ouvrir la console avec iceweasel (firefox) ?
    Même question avec chrome ?
  2. Mettez l'ancien code en commentaire.
    Ecrivez un code permettant d'afficher, toujours dans la console, les chiffres de 0 à 99 inclus (il n'est pas nécessaire de faire une fonction pour l'instant).
    Note : par pitié, pas de copier/coller !
  3. Changez le code pour afficher les chiffres de 0 à 99 sauf les multiples de 3 (c-a-d sauf ceux qui sont divisibles par 3).
  4. Commentez l'ancien code et écrire une fonction (que vous nommerez de façon claire et intelligente) prenant un entier en paramètre et affichant sur la console les nombres de 0 à cet entier (non compris). Puis écrivez un code de test de la fonction avec les valeurs 3 et 12.

Exercice 2 - Damier

On cherche à faire un damier avec des cases blanches et noires et la diagonale affichée en rouge.

  1. Recopiez la page Web exo2.html dans votre répertoire tp5.
    Expliquez ce que fait la première ligne du script : 
    "use strict";
  2. Expliquez ce que fait l'appel document.write().
  3. Ajoutez une fonction drawCheckerLine qui affiche une ligne de 5 cases grâce à la méthode drawCheckerCell.
    Note: Le plus simple est de mettre les 5 boites collées à gauche à l'intérieur d'une grosse boite qui correspondra à la ligne.
  4. Ajoutez une fonction drawCheckerBoard pour afficher un carré de 5 cases par 5 cases.
  5. Modifiez la fonction drawCheckerBoard pour que dans la page web, on puisse afficher un carré de 5 par 5 puis un carré de 3 par 3. Vérifiez que l'affichage se fait correctement.
    Note: attention au float en CSS !
  6. Complétez le style de la balise style du document pour définir une case noire et une case blanche et utilisez ce style pour les cases du damier.
    Note: une balise HTML peut avoir plusieurs classes CSS !
    Astuce: la somme des coordonnées des cases modulo 2 (le reste de la division par 2) vous indique s'il s'agit d'une case blanche ou d'une case noire.
  7. Ajoutez un style de case rouge puis ajoutez un dernier paramètre à drawCheckerBoard pouvant être vrai ou faux. Si le paramètre est vrai, alors il faudra afficher la diagonale en rouge.
  8. Testez en affichant la diagonale du premier carré (pas celle du second).

Exercice 3 - Pour les plus balèzes... (Optionnel)

Une petite question supplémentaire.
On peut améliorer la 'beauté' du code, et le rendre un peu plus professionnel, en évitant de passer en paramètre un booléen pour dire s'il faut dessiner ou non la diagonale.
L'idée est d'envoyer à la fonction drawCheckerBoard une fonction d'affichage qui choisi la couleur en fonction d'une coordonnée passé en paramétre. Puis d'appeler drawCheckerBoard deux fois, une fois avec une fonction qui colore la diagonale et autre une fois avec une fonction qui ne colore pas la diagonale.
