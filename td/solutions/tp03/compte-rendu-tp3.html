Exercice 2 : La ferme des animaux

  1. Voilà, c'est fait !
  2. J'ai ajouté 
    div img {
  width: 180px;
}
    entre les balises <style> de l'en-tête du document.
  3. J'ai ajouté 
    * {
  float: left;
}
div div {
  clear: both;
}
    entre les balises <style> de l'en-tête du document.
  4. Les boîtes de droite passent à la ligne quand la fenêtre du navigateur n'est pas assez large !
  5. Quand j'écris 
    * {
  float: right;
}
    au lieu de 
    * {
  float: left;
}
    les images sont maintenant alignées à droite (et l'image 1 est la plus à droite, alors que c'était la plus à gauche).
  6. J'ai ajouté 
    color: white;
background-color: black;
width: 100%;
text-align: center;
padding-top: 5px;
padding-bottom: 5px;
    dans les informations de style capturées par le sélecteur 
    div div {
}
    de l'en-tête du document.
  7. Pour sélectionner uniquement les boîtes externes mais pas les boîtes internes (contenant les titres), j'ai ajouté 
    div {
  margin-left: 5px;
  margin-right: 5px;
}
div * {
  margin-left: 0px;
  margin-right: 0px;
}
    entre les balises <style> de l'en-tête du document.
    Une alternative raisonnable, et sans doute plus adéquate, eût été de conférer la classe "title" à chaque boîte contenant juste un titre et la classe "container" à chaque boîte contenant une photo et une boîte de titre. De la sorte, on pourrait facilement appliquer un style aux seules boîtes de titre ou bien aux seules boîtes englobantes.

    entre les balises <style> de l'en-tête du document.
  10. J'ai ajouté

    dans les informations de style capturées par le sélecteur

}
    de l'en-tête du document.
