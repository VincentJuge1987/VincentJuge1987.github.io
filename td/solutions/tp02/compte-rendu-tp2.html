Exercice 3 : Mon compte-rendu est trop stylé !

  1. J'ai utilisé la balise <h2 style="color: orange;">.
  2. J'aurais pu écrire, au choix, 
    <style>
 * {
   font-family: sans-serif;
 }
</style>
    ou bien 
    <style>
 body {
   font-family: sans-serif;
 }
</style>
    dans l'en-tête du document.
  3. J'ai ajouté 
    h1 {
  font-size:24pt;
}
h2 {
  font-size:18pt;
}
h3 {
  font-size:16pt;
}
    entre les balises <style> de l'en-tête du document.
  4. J'ai ajouté 
    h2 {
  color: navy;
}
    entre les balises <style> de l'en-tête du document.
  5. J'ai utilisé la balise <h1 class="centered"> puis j'ai ajouté 
    .centered {
  text-align: centered;
}
    entre les balises <style> de l'en-tête du document.
  6. J'ai écrit <div class="right"> Vincent Jugé </div> dans le corps du document, puis j'ai ajouté 
    .right {
  text-align: right;
}
    entre les balises <style> de l'en-tête du document.
  7. J'ai ajouté 
    a {
  color: red;
}
a:hover {
  color: teal;
}
    entre les balises <style> de l'en-tête du document.
